# odoo-maventa

Odoo module for sending electronic invoices (Finvoice) through the Maventa operator's API.

## Repository

- **GitHub**: https://github.com/Artkork/odoo-maventa
- **Clone (HTTPS)**: `git clone https://github.com/Artkork/odoo-maventa.git`
- **Clone (SSH)**: `git clone git@github.com:Artkork/odoo-maventa.git`

## Branches

- `main` — stable production-ready code

## Modules

- [`maventa_finvoice`](maventa_finvoice/README.md) — Finvoice invoice sending through Maventa operator

## Quick Start

```bash
git clone https://github.com/Artkork/odoo-maventa.git
cd odoo-maventa
pip install -r requirements.txt
```

Then copy the `maventa_finvoice` directory to your Odoo addons path and install the module from the Odoo Apps menu.
