Maventa Finvoice for Odoo 19

Electronic Invoice Transmission via Maventa Operator

This module enables seamless sending of invoices in Finvoice format through the Maventa operator's secure network. Designed for Odoo 19 Enterprise, it provides full integration with Odoo's accounting system.

Key Features:

  • Finvoice Format Support - Generate standard Finvoice 3.2.2 XML format
  • Maventa API Integration - Direct integration with Maventa transmission network
  • Multi-Recipient Sending - Send to multiple customers simultaneously
  • Status Tracking - Real-time delivery status from Maventa
  • Automatic Transmission - Optional auto-send when invoices are posted
  • Error Management - Detailed error logging and troubleshooting
  • Sandbox Support - Test mode for Maventa's sandbox environment
  • Audit Trail - Complete transmission history and XML content preservation

User Interface Enhancements:

  • Send Finvoice button on customer invoices
  • Status indicator showing transmission state
  • Separate Finvoice page in invoice form
  • Transmission logs with full details
  • Configuration interface for Maventa settings

Requirements:

  • Odoo 19 Enterprise Edition
  • Active Maventa account with API credentials
  • Python packages: requests, lxml (auto-installed)

Getting Started:

  1. Install the module
  2. Configure Maventa API credentials in Settings
  3. Test connection
  4. Start sending invoices as Finvoice

For detailed documentation see the README.md file included in the module. For Maventa support, contact Maventa directly.
